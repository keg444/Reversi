# リバーシを作る
# How to use
1. Go to the forked directory and run the following command:
```
python3 osero.py
```
2. Enter the location (marked as "?") of the stone.
   ex)32
3. Switch players and repeat the same action.
4. The game ends when a player can no longer place stones or when the board is filled with stones.
5. Enter "y" to play the game again or "n" to quit.(You can quit the game at any time by pressing "q" or "Q".)
